# Dart & Flutter Lehrplan

Strukturierter Selbstlernkurs f&uuml;r Dart und Flutter -- von den Sprachgrundlagen bis zur fertigen App.

## Zielgruppe

Entwickler mit Vorkenntnissen in C++, JavaScript/HTML/CSS und Python. Grundlegende Programmierkonzepte (Variablen, Schleifen, Funktionen, OOP-Basics) werden als bekannt vorausgesetzt.

## Aufbau

Der Lehrplan ist in **2-Stunden-Lerneinheiten** gegliedert. Jede Einheit besteht aus Lehrstoff, Ressourcen, einer &Uuml;bung und einer L&ouml;sung.

### Dart & Flutter (44 Einheiten, ~88 Stunden)

| Block | Schwerpunkt | Einheiten | Status |
|-------|-------------|-----------|--------|
| **1** | [Dart -- Die Sprache](block_1_dart/) | 10 | fertig |
| **2** | [Flutter -- Grundlagen](block_2_flutter_grundlagen/) | 10 | fertig |
| **3** | Flutter -- Fortgeschritten | 12 | geplant |
| **4** | Profi-Themen & Abschlussprojekt | 12 | geplant |

### Dart Backend (geplant)

Ein weiterer Lehrplan f&uuml;r Dart-Backend-Entwicklung (z.B. mit `shelf`, `dart_frog` oder `serverpod`) ist in Vorbereitung.

## Struktur einer Lerneinheit

| Phase | Dauer | Inhalt |
|-------|-------|--------|
| Theorie | 45 min | Lehrstoff lesen, Code-Beispiele nachvollziehen |
| Praxis | 60 min | &Uuml;bung bearbeiten |
| Review | 15 min | L&ouml;sung vergleichen, Ressourcen bei Bedarf |

Der vollst&auml;ndige Lehrplan mit allen Einheiten findet sich in [Lehrplan.md](Lehrplan.md).

## Voraussetzungen

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- VS Code mit Dart & Flutter Extensions (oder Android Studio)
- F&uuml;r Block 1 reicht [DartPad](https://dartpad.dev)

```bash
flutter doctor       # Installation pr&uuml;fen
flutter create app   # Neues Projekt erstellen
flutter run          # App starten
```
