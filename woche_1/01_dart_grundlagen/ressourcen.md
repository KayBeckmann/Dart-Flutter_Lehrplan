# Modul 1: Ressourcen — Dart Syntax & Grundlagen

## Offizielle Dokumentation

- **Dart Language Tour** (Pflichtlektüre!)
  https://dart.dev/language
  Umfassende Einführung in alle Sprachfeatures von Dart. Sehr gut strukturiert und mit vielen Beispielen.

- **Dart API-Referenz**
  https://api.dart.dev/stable/
  Vollständige Dokumentation aller Standardbibliotheken (`dart:core`, `dart:math`, `dart:io`, etc.).

- **Effective Dart** (Stil- und Best-Practice-Guide)
  https://dart.dev/effective-dart
  Vier Abschnitte: Style, Documentation, Usage und Design. Sollte jeder Dart-Entwickler kennen.

- **Dart Style Guide**
  https://dart.dev/effective-dart/style
  Naming Conventions, Formatierung und Codeorganisation.

## Interaktive Lernumgebungen

- **DartPad** (Online-Editor)
  https://dartpad.dev
  Dart-Code direkt im Browser schreiben und ausführen. Perfekt zum Experimentieren und für die Übungen dieses Moduls.

- **Dart Cheatsheet Codelab**
  https://dart.dev/codelabs/dart-cheatsheet
  Interaktives Codelab, das die wichtigsten Dart-Features in Kurzform zeigt.

- **Intro to Dart for Java Developers**
  https://dart.dev/codelabs/dart-cheatsheet
  Obwohl für Java-Entwickler geschrieben, ist es auch für C++-Entwickler hilfreich, da viele Konzepte übertragbar sind.

## Bücher

- **Dart Apprentice** (raywenderlich.com / Kodeco)
  https://www.kodeco.com/books/dart-apprentice
  Umfassendes Einsteigerbuch, das systematisch alle Dart-Grundlagen abdeckt.

- **Dart in Action** (Manning Publications)
  Für Entwickler, die bereits eine andere Sprache beherrschen.

## Tools und Setup

- **Dart SDK installieren**
  https://dart.dev/get-dart
  Installationsanleitung für alle Plattformen (Linux, macOS, Windows).

- **VS Code mit Dart-Extension**
  https://marketplace.visualstudio.com/items?itemName=Dart-Code.dart-code
  Empfohlener Editor mit exzellentem Dart-Support (Autocomplete, Debugging, Refactoring).

- **IntelliJ IDEA / Android Studio mit Dart Plugin**
  https://plugins.jetbrains.com/plugin/6351-dart
  Alternative IDE mit starker Dart-Integration.

## Vergleichs-Ressourcen (für Umsteiger)

- **Dart for C++ Developers**
  Kein dedizierter offizieller Guide, aber die Language Tour deckt alle relevanten Unterschiede ab.

- **Dart for JavaScript Developers**
  https://dart.dev/guides/language/coming-from/js-to-dart
  Offizieller Vergleich zwischen JavaScript und Dart.

- **Dart for Python Developers**
  Die Effective Dart Guides erklären viele Konventionen, die sich von Python unterscheiden (z.B. camelCase statt snake_case).
