# Modul 10: Ressourcen -- HTTP, REST-APIs & JSON

## Packages

- **http Package auf pub.dev**
  https://pub.dev/packages/http
  Das Standard-HTTP-Package fuer Dart/Flutter. Leichtgewichtig und einfach zu verwenden.

- **dio Package auf pub.dev**
  https://pub.dev/packages/dio
  Fortgeschrittenes HTTP-Package mit Interceptors, Cancel Tokens, FormData und mehr.

- **json_annotation Package**
  https://pub.dev/packages/json_annotation
  Annotationen fuer die automatische JSON-Serialisierung.

- **json_serializable Package**
  https://pub.dev/packages/json_serializable
  Code-Generierung fuer fromJson/toJson. Spart viel Boilerplate.

- **build_runner Package**
  https://pub.dev/packages/build_runner
  Build-System fuer Code-Generierung in Dart.

## Test-APIs

- **JSONPlaceholder**
  https://jsonplaceholder.typicode.com
  Kostenlose Fake-REST-API zum Testen. Perfekt fuer Lernzwecke. Bietet Users, Posts, Comments, Albums, Photos, Todos.

- **ReqRes**
  https://reqres.in
  Weitere kostenlose Test-API mit Authentifizierung-Simulation.

- **httpbin.org**
  https://httpbin.org
  Nuetzlich zum Testen von HTTP-Methoden, Headers, Status Codes etc.

- **Dog API**
  https://dog.ceo/dog-api/
  Spassige API mit Hundebildern. Gut fuer Bildlade-Uebungen.

## Tools zum Testen von APIs

- **Postman**
  https://www.postman.com
  Der Klassiker unter den API-Test-Tools. Grafische Oberflaeche zum Senden von Requests.

- **Thunder Client (VS Code Extension)**
  https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client
  Leichtgewichtige Postman-Alternative direkt in VS Code.

- **Insomnia**
  https://insomnia.rest
  Elegantes API-Test-Tool als Desktop-Anwendung.

- **curl (Kommandozeile)**
  Auf den meisten Systemen vorinstalliert. Nuetzlich fuer schnelle Tests:
  ```bash
  curl https://jsonplaceholder.typicode.com/users/1
  ```
